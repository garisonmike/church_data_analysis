# Customising Church Analytics

This document covers visual customisation — specifically the app icon. For
broader forking steps (app name, application ID, signing keystore, CI setup)
see `docs/FORKING.md`.

---

## App Icon

### Source file

The single source of truth for the app icon is:

```
assets/images/app_icon.png
```

Replace this file with your church's logo. All platform-specific icon sizes
(Android mipmap densities, Windows `.ico`, web favicon, Linux PNG set) are
generated automatically from this one file by `flutter_launcher_icons`.

### Requirements

| Requirement | Detail |
|---|---|
| Dimensions | 1024 × 1024 pixels (square) |
| Format | PNG |
| Transparency | Supported — Android adaptive icon will apply a coloured background layer |
| File name | Must stay `app_icon.png` — `pubspec.yaml` references it by path |

> **Tip:** Keep a copy of your original artwork at full resolution outside the
> repository. The file committed here will be scaled down by the generator and
> is not a lossless archive of your source artwork.

### How to regenerate icons after replacing the file

```bash
dart run flutter_launcher_icons
```

This reads the `flutter_launcher_icons` section of `pubspec.yaml` (already
fully configured) and writes all platform icon assets. You do not need to
touch any platform-specific folder by hand.

Commit all generated files — do **not** commit only the source PNG and skip
this step, or the installed app will show the old icon on some platforms.

### Rebuild the app after icon generation

```bash
flutter build apk --release          # Android
flutter build windows --release      # Windows
```

A running dev build (`flutter run`) will not pick up launcher icon changes
until a full rebuild is done.

### Android adaptive icon background colour

Android 8+ shows a background colour behind the icon shape. The adaptive icon
configuration in `pubspec.yaml` uses a plain white background by default:

```yaml
flutter_launcher_icons:
  image_path: "assets/images/app_icon.png"
  android: true
  # adaptive_icon_background: "#FFFFFF"   # uncomment to set a custom colour
```

Uncomment `adaptive_icon_background` and set a hex colour that complements
your logo, then re-run `dart run flutter_launcher_icons`.

### Starter template

If you are designing an icon from scratch, a good starting point is a
1024 × 1024 white canvas with your church's initials or symbol centred within
the inner 512 × 512 pixels (the "safe zone" that is always visible regardless
of the Android icon mask shape).

## Other visual assets

| Asset | Location | Notes |
|---|---|---|
| Splash / launch background | `android/app/src/main/res/drawable/launch_background.xml` | Edit the colour or add a centred image |
| Web favicon | Regenerated by `flutter_launcher_icons` from `app_icon.png` | No manual edit needed |
| Windows taskbar icon | Regenerated by `flutter_launcher_icons` as `windows/runner/resources/app_icon.ico` | No manual edit needed |
